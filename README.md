# RAPBS Yayasan Al-Wathoniyah9

Sistem Rencana Anggaran Pendapatan dan Belanja Sekolah (RAPBS) Yayasan Al-Wathoniyah9.

## Tentang Aplikasi

Aplikasi ini dirancang untuk mengelola keuangan sekolah, termasuk perencanaan anggaran, pencatatan pemasukan dan pengeluaran, serta pelaporan keuangan.

## Fitur Utama

- Manajemen User dan Role (RBAC)
- Manajemen Profil Pengguna
- Pencatatan RAPB Sekolah
- Laporan Keuangan
- Audit Log

## Memulai

Ikuti langkah-langkah instalasi standar Laravel untuk menjalankan aplikasi ini.
