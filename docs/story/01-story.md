This is the list feature that this filament project implements.
1. I want to the login, register, forgot password from filament enabled in this project.
2. I want to have role based access control using filament-shield library.
3. I want to have user profile feature implemented using filament-breezy package.
4. I want to have ability to impersonate other users using filament-impersonate package.
5. I want to have ability to allow user to login using social login providers using filament-socialite package.
6. I want this project to use spatie media library to manage files and images uploaded by users.
